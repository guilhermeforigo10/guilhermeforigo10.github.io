# GM Troféus e Medalhas — Landing page

Site estático (HTML/CSS/JS). Sem build, sem dependência de servidor.

## Publicar no GitHub Pages
1. Crie um repositório público (ex: `gm-trofeus`).
2. Suba TODO o conteúdo desta pasta na raiz do repo.
3. Settings > Pages > Source: branch `main` / `/ (root)` > Save.
4. ~1 min depois: `https://SEU-USUARIO.github.io/gm-trofeus/`

## Publicar na Hostinger
Suba o conteúdo desta pasta dentro de `public_html` (File Manager ou FTP).
O arquivo `index.html` é a home.

## Estrutura
- index.html — página principal
- /assets — logos, vídeo da medalha (hero), fotos do portfólio
- /frames-medal — frames da medalha (seção Garantia)
- /js — medal-stage.js, medal-section.js, medal.js, ui.js
